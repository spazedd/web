# XP Web Desktop

## Folders & Files
- `index.html`: Main file – open this in browser.
- `assets/icons/`: 9 PNG files (exact names above).
- `assets/wallpaper/bliss.jpg`: The wallpaper (download from https://cdn.neowin.com/news/images/uploaded/2023/06/1686292349_windows_xp_bliss_wallpaper_4k.jpg).

## How to Run
1. Download icons & bliss.jpg as per instructions.
2. Open index.html.

All apps open fully – IE browses, Paint draws, etc. Wallpaper change is optional in Settings.
